What is CFW?

CFW is a lot like Linux, in the way Linux has distros. There are a crap ton of Linux distros, however only some are used. Same goes for PS3.

The main CFW most used is REBUG, or EVILNAT. Both are great, and they have differences:

  • REBUG has always had DEX versions
  • EVILNAT has more features, however its less mature than REBUG
  • REBUG has OtherOS, DEX support, but its on older FW versions only, so SEN access can be annoying to get working.

    • EVILNAT (otherwise known as Nathan) has started testing his "PEX/DPEX" CFW. This CFW has more XAI features, as well as DEX. This is the only 4.89 DEX CFW, ever since the last DEX OFW release, 4.84

    When 4.89.3 releases, I will put a download link here.

    Differences between CFW and PS3HEN

  • PS3HEN uses HFW, but PS3HEN must be installed after HFW
  • CFW doesnt have this problem. For the first install, just patch the console, then install CFW.
  • PS3HEN must be enabled after every reboot (wether it be a LV2 reboot, or hardware reboot)
  • CFW uses either COBRA or MAMBA (MAMBA is a different version of COBRA, its essentially the same but PS3HEN uses it for base as its more up to date)
  • CFW is more stable and works with more homebrews.
